# Preguntas frecuentes — Intake

## Sobre el producto

**¿Qué es Intake?**
Un recepcionista automático para WhatsApp. Atiende a tus clientes, recoge los datos
de cada trabajo de forma conversacional y te avisa cuando algo está listo para que
lo revises. Incluye un panel web para gestionar todo.

**¿Para qué negocios sirve?**
Para negocios que reciben pedidos por WhatsApp: tapicería, paquetería,
reparaciones y servicios en general. Si sueles pedir "una foto y las medidas",
Intake encaja.

**¿Necesito saber de tecnología?**
No. Si usas WhatsApp, puedes usar Intake. El registro y la configuración están
pensados para cualquier persona.

**¿El bot entiende fotos y audios?**
Sí. Interpreta las fotos (por ejemplo, del mueble) y transcribe los audios para
sumarlos a la ficha del trabajo.

**¿Puedo atender yo una conversación?**
Sí. Puedes pausar el bot en un contacto y tomar tú la conversación cuando quieras.

## Sobre WhatsApp

**¿Usa mi propio número?**
Sí, el de tu negocio. Lo conectas escaneando un código QR (igual que WhatsApp
Web) y puedes desvincularlo cuando quieras.

**¿Es la conexión oficial de WhatsApp?**
Hoy usamos la conexión por "dispositivos vinculados" (la de WhatsApp Web), no la
API oficial de WhatsApp Business. Funciona muy bien para la mayoría de negocios.
Como depende de WhatsApp, en casos excepcionales el número podría desconectarse o
ser limitado por sus políticas; por eso te avisamos si el bot se cae y trabajamos
en reconectarlo. Lo explicamos con detalle en el aviso de uso de WhatsApp.

**¿Qué pasa si el número se desconecta?**
Te avisamos. Muchas veces reconecta solo; si no, solo tienes que volver a escanear
el QR desde el panel. Tus datos y conversaciones no se pierden.

**¿Van a tener la API oficial de WhatsApp?**
Es una mejora que estamos evaluando para quienes necesiten máxima estabilidad.

## Privacidad y seguridad

**¿Qué hacen con los datos de mis clientes?**
Los procesamos para prestarte el servicio (atender mensajes y armar el intake),
siguiendo tus instrucciones. Tú eres el responsable de esos datos y nosotros el
encargado. No los vendemos ni los usamos para entrenar modelos. Ver la Política de
Privacidad y el DPA.

**¿Están seguros?**
Usamos cifrado en tránsito, contraseñas protegidas, aislamiento entre cuentas y
copias de seguridad. Ningún sistema es infalible, pero tomamos medidas serias.

**¿Puedo exportar o borrar mis datos?**
Sí. Puedes solicitar la exportación o el borrado de los datos de tu cuenta.
